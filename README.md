# YAG v1

Простой Bash-скрипт для сбора информации о системе и генерации отчёта в HTML. Вторая версия, специально по заказу. 

## Возможности

- 🖥️ **Система** – ядро, аптайм, дата;
- 🛠️ **Железо** – CPU, память, диски, PCI;
- 🌐 **Сеть** – IP, маршруты, DNS;
- 📦 **Пакеты** – список установленных пакетов;
- 🛡️ **Антивирусы** – ClamAV, Kaspersky, Dr.Web, ESET и др.;
- 🎮 **NVIDIA** – информация о видеокарте (если есть драйверы);
- ⚙️ **Службы и процессы** – запущенные сервисы и процессы;
- 👥 **Пользователи и группы** – локальные учётные записи;
- 📜 **Логи** – системные логи и dmesg.
